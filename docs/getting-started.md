# 🚀 Getting Started

## Pré-requisitos

- Node.js 22+
- MySQL 8+
- Git

---

## Clonar

```bash
git clone https://github.com/PedroLung/rede-social-toughts.git

cd rede-social-toughts
```

---

## Instalar dependências

```bash
npm install
```

---

## Configurar banco

Crie um banco MySQL.

```
thoughts
```

Configure a conexão.

---

## Executar

```bash
npm start
```

---

!!! success

    A aplicação estará disponível em http://localhost:3000
