# 🔥 Runbook

## Aplicação não inicia

Verifique:

```bash
npm install
```

Depois:

```bash
npm start
```

---

## Banco indisponível

Verifique:

- MySQL iniciado
- Credenciais
- Banco criado

---

## Sessão não funciona

Confira:

- express-session
- cookie-parser

---

!!! danger

    Nunca reinicie o banco de produção sem backup.
