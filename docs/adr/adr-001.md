# ADR-001

# Utilização do Sequelize

## Status

Aceito

---

## Contexto

Era necessário simplificar a comunicação com o banco de dados.

---

## Decisão

Foi escolhido o Sequelize como ORM.

---

## Consequências

### Positivas

- Menos SQL manual
- Migrations
- Models

### Negativas

- Pequena perda de performance
