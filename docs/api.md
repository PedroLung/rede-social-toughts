# 📦 API

A aplicação utiliza rotas Express.

## Usuários

| Método | Endpoint | Descrição |
|---------|----------|-----------|
| POST | /login | Login |
| POST | /register | Cadastro |
| GET | /logout | Logout |

---

## Pensamentos

| Método | Endpoint | Descrição |
|---------|----------|-----------|
| GET | /thoughts | Lista pensamentos |
| POST | /thoughts/add | Novo pensamento |
| POST | /thoughts/edit | Edita pensamento |
| POST | /thoughts/remove | Remove pensamento |

---

!!! note

    Em versões futuras recomenda-se migrar para uma API REST documentada com OpenAPI.
